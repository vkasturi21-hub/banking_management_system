Banking Management System
Project Overview:This project is a Python-based Banking Management System that manages customer accounts, banking transactions, and financial reporting. It also performs data analysis and visualization to provide business insights into customer behavior, account balances, and transaction patterns.

The project demonstrates practical applications of Python programming, data processing, and business intelligence concepts in a banking environment.
TECHNOLOGIES: python,SQLite,SQLAIchemy,Pandas
Security & Cryptography:Passlib & Bcrypt
Visual Analytics:Matplotlib & Seaborn
Software Quality & QA: Unittest
